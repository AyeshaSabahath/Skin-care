# \# Derma-web

# 

# A skin disease detection web application built using Machine Learning and Deep Learning.

# 

# \## Features

# \- Skin disease image classification

# \- User-friendly web interface

# \- Model prediction with confidence score

# \- Training and testing pipeline

# \- Dataset support

# 

# \## Technologies Used

# \- Python

# \- TensorFlow / Keras

# \- Flask

# \- HTML / CSS / JavaScript

# 

# \## Project Structure

# ```bash

# Derma-web/

# │── app.py

# │── Trainmodel.py

# │── dataset/

# │── templates/

# │── static/

# │── README.md

# │── requirements.txt

# ```

# 

# \## Dataset

# The dataset contains skin disease images organized by class folders.

# 

# Example:

# ```bash

# dataset/

# │── acne/

# │   ├── img1.jpg

# │   ├── img2.jpg

# │── eczema/

# │   ├── img1.jpg

# ```

# 

# \## How to Run

# ```bash

# pip install -r requirements.txt

# python app.py

# ```

# 

# \## Model Training

# ```bash

# python Trainmodel.py
